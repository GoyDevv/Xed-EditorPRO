================================================================================
NEW in 4.1.9
================================================================================
- core/main/src/main/java/com/rk/ai/AiTaskStore.kt (new)
  Process-wide task plan mirrored to Compose state: AiTask(text, done); set(list), add(text),
  complete(1-based index), clear(), render() (returns "1. [x] …" text for the model).
- core/main/src/main/java/com/rk/ai/AiTools.kt (edited) — two new tools:
    • set_tasks(tasks: string[]) — create/replace the plan.
    • complete_task(index)       — mark a step done (1-based).
  These (and the read-only tools) are auto-allowed (no permission prompt).
- core/main/src/main/java/com/rk/ai/AiViewModel.kt (edited)
  • askPermission auto-allows set_tasks/complete_task/read_file/list_dir/search_text/glob_files.
  • System prompt now instructs the model to call set_tasks first for multi-step work and
    complete_task as it finishes each step.
  • newSession() clears the task list.
- core/main/src/main/java/com/rk/ai/AiTab.kt (edited)
  • A "Tasks" panel above the messages shows the live plan (✓ done in primary color, • pending).
- app/build.gradle.kts: 4.1.8 -> 4.1.9 (versionCode 115 -> 116).
- phases.md: task tools noted.
