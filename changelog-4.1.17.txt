Xed-Editor PRO — Changelog
Version 4.1.17 (versionCode 124)

AI Agent — removed Kiro; reworked Gemini (Google login): no API key, account status, live streaming,
model list, method shown in the chat header, and sign-out.

>>> BUILD NOTE: does NOT build locally (JDK 17; needs JDK 21 + SDK/NDK). Verified by GitHub Actions CI.
>>> Gemini web uses Google's undocumented internal API; needs on-device testing.

================================================================================
REMOVED
================================================================================
- Kiro is gone entirely (KiroAuth/KiroClient/KiroSetup/KiroSetupUi deleted; KIRO provider, routing,
  prefs, and all UI entry points removed) — there was no reliable way to run kiro-cli on device.

================================================================================
GEMINI (Google login) — reworked
================================================================================
- No API key option: selecting "Gemini (Google login)" now shows only "Sign in with Google"
  (AiTab.AddKeyDialog) — no key field.
- Account + availability: GoogleLoginDialog is now an account screen — when signed in it shows the
  Google account, checks whether Gemini is available (some accounts must accept terms first: it tells
  you to open Gemini and finish, then "Check again"), and offers "Sign out / delete login".
  The email is auto-detected best-effort (GeminiWebClient.fetchTokens).
- Live streaming (fixes "keeps thinking / did nothing"): GeminiWebClient.chatStream now reads the
  batchexecute stream line-by-line and emits text via onDelta in real time (instead of waiting for the
  whole body, which could hang). If nothing parses, it now shows an explicit error instead of hanging.
- Model list: the picker shows real Gemini models (2.5 Flash/Pro, 2.0 Flash, 1.5 Flash/Pro) instead
  of just "gemini".
- Chat header: shows the chat name AND the method in use, e.g. "New chat" / "Gemini (Google login) ·
  you@gmail.com" or "OpenAI · API key" (AiViewModel.methodLabel + AiTab top bar).
- Sign out: from the account screen and from Settings → AI Agent → "Gemini (Google login)".
- Cookies persist (AiPrefs.geminiPsid/geminiPsidts/geminiAccount).

- app/build.gradle.kts: 4.1.16 -> 4.1.17 (versionCode 123 -> 124).

Note: consumer Gemini has no native tool-calling, so tools use a text protocol (<tool_call>{...}</tool_call>)
parsed back into real calls. The Gemini API-key provider remains the reliable, tool-native option.
