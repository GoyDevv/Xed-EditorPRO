NEW in 4.1.5
================================================================================
- core/main/src/main/java/com/rk/ai/AiSettingsScreen.kt (new)
  Settings screen to manage the AI agent (PreferenceLayout/PreferenceGroup/SettingsItem pattern,
  matching GitSettings). Sections:
    • Provider  — pick provider (OpenRouter/OpenAI/Gemini/Custom); Base URL (Custom only); API key
                  (input dialog with "Verify & save" → AiClient.listModels, or plain Save).
    • Model     — set the default model id.
    • Permissions — "Reset tool permissions" sets every tool back to ASK.
  Reads/writes AiPrefs directly; a `rev` counter refreshes descriptions after edits.
- core/main/src/main/java/com/rk/activities/settings/SettingsRoutes.kt: added Ai ("ai_settings").
- core/main/src/main/java/com/rk/activities/settings/SettingsNavHost.kt: routes Ai → AiSettingsScreen().
- core/main/src/main/java/com/rk/settings/SettingsScreen.kt: added an "AI Agent" category
  (icon drawables.bolt) in the main Settings list, after Keybindings.
- app/build.gradle.kts: 4.1.4 -> 4.1.5 (versionCode 111 -> 112).
- phases.md: AI Settings screen marked done.
