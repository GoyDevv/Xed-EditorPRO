================================================================================
NEW in 4.1.3
================================================================================
- core/main/src/main/java/com/rk/ai/AiViewModel.kt (edited)
  • Added runJob (the agent turn's coroutine) and stop(): cancels runJob, clears the permission gate
    (completes any pending request as denied), and sets busy=false. send() now stores runJob.
- core/main/src/main/java/com/rk/ai/AiTab.kt (edited)
  • The composer's send button becomes a Stop button (drawables.stop → vm.stop()) while the agent is
    running, so you can interrupt a turn.
- app/build.gradle.kts: 4.1.2 -> 4.1.3 (versionCode 109 -> 110).
- phases.md: cancel/stop marked done (with the note that true in-flight OkHttp cancel is still TODO).

Known limitation: stop() cancels the coroutine (loop halts, UI unblocks) but the in-flight streaming
HTTP read continues in the background until the response ends; its result is then discarded. Proper
fix = hold the okhttp3.Call and cancel() it. Documented in phases.md.
