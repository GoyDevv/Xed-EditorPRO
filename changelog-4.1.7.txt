================================================================================
NEW in 4.1.7
================================================================================
- core/main/src/main/java/com/rk/ai/AiTools.kt (edited) — three more tools (schema + describe +
  execute), permission-gated like the rest:
    • edit_file(path, old_string, new_string, replace_all?) — targeted str-replace edit (the Claude
      Code "Edit" tool); errors if old_string isn't found.
    • glob_files(pattern, path?) — find files by glob (* ? and ** ); skips heavy dirs, caps at 200.
    • create_dir(path) — mkdirs.
  Full toolset now (AiTools.names): read_file, write_file, edit_file, create_dir, list_dir,
  glob_files, search_text, run_command, delete_file, move_file.

- core/main/src/main/java/com/rk/ai/AiViewModel.kt (edited) — clearCurrent() and lastAssistant() for
  the slash commands.

- core/main/src/main/java/com/rk/ai/AiTab.kt (edited) — chat **slash /commands**: typing "/" shows a
  live suggestion list above the composer; commands run LOCALLY (not sent to the model):
    /new (new chat), /clear (clear chat), /model (open model picker), /stop (cancel turn),
    /copy (copy last reply), /help (dialog listing all commands).
  See Composer → slashCommands + runSlash; the send button routes "/…" through runSlash.

- app/build.gradle.kts: 4.1.6 -> 4.1.7 (versionCode 113 -> 114).
- phases.md: tools + slash commands updated.
