NEW in 4.1.8

- core/main/src/main/java/com/rk/ai/AiClient.kt (edited)
  • Holds the in-flight okhttp3.Call (currentCall) and adds cancel(); chatStream wraps execute() in
    try/finally so Stop actually aborts the streaming network request mid-flight.
- core/main/src/main/java/com/rk/ai/AiViewModel.kt (edited)
  • stop() now calls AiClient.cancel() too; the run job ignores CancellationException (Stop no longer
    shows a spurious error).

    (e.g. "Run: npm test", "Edit src/Main.kt") via AiAgentService.start().

  • AssistantContent: assistant replies now render fenced ``` code blocks as monospace cards with a
    Copy button (drops the optional language tag line); normal text otherwise.

- phases.md: Phase 3 items updated; added a note on the agent-terminal design.
